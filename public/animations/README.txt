Letakkan animasi Lottie di folder ini:

reading.json
   Animasi "mahasiswa membaca buku" format Lottie JSON.
   Tampil di section "Tentang" landing page (lottie-player, autoplay + loop).
   Unduh gratis di https://lottiefiles.com (cari "reading book" / "student reading"),
   pilih format "Lottie JSON", lalu simpan sebagai reading.json di folder ini.

   Jika file belum ada, area animasi akan kosong (tidak error). Layout tetap rapi.
